== Description ==

WP Easy Popup Plugin  is world renowned as the most beautiful and easy to create popups for different pages in the market.
We can create more popups with different-2 styles and attractive for different-2 pages.
The WP Easy Popup is completely free and totally open source, and there is literally no better way to make your website look totally stunning.

You can also customize setting for following features:

* You can add a header or heading section on the top of plugin.
* You can customize the Background color of header section.
* You can set the color and the font size of text in header section as well.
* You can add image to the Popup.
* You can get Popup with or without image.
* You can design the close button according to your color-combination of Popup by changing its background-color and color.
* You can set the Popup display time.
* You can set the time to remove the popup automatically as well. 
* You can customize Width of Popup box.
* You can also customize Background color and Text color also.
* You can also customize Popup display days.
* You can also hide Popup box on mobile devices.
* You can also hide popup box for logged-in users.

== Installation ==

Installation consists of following steps:

1. Upload "wp-easy-popup" to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create many popups and add contents from backend and style them.
